# API REST - Metaindústria

## Descrição

API REST desenvolvida em Spring Boot para gerenciamento de alertas industriais gerados por sistemas de visão computacional.

A aplicação permite cadastrar, listar, atualizar e remover alertas relacionados à segurança industrial.

## Entidade

AlertaIndustrial

## Tecnologias

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- Banco H2
- Maven

## Endpoints

POST /alertas

GET /alertas

GET /alertas/{id}

PUT /alertas/{id}

DELETE /alertas/{id}

## Como executar

1. Clonar o projeto
2. Abrir na IDE
3. Executar a classe principal
4. Acessar:
http://localhost:8080

## Console H2

http://localhost:8080/h2-console
